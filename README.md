# Sable Dark Modern

A refined dark theme for Zed based on One Dark Pro Max, with VS Code Dark Modern inspired panels, borders, and selections.

## Preview

Preview images will be added later.

## Installation

Install from Zed's extension registry after publication, or install it locally as a dev extension:

1. Open the Zed command palette.
2. Run `zed: install dev extension`.
3. Select this repository directory.
4. Choose `Sable Dark Modern` as your theme.
